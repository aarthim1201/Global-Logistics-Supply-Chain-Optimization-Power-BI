# Global Logistics and Supply Chain Optimization

## Project Overview
This project focuses on optimizing logistics and supply chain operations using Power BI. It provides insights into delivery performance, shipping costs, and inventory management.

## Key Insights
- *On-Time Delivery:* Analyzed delivery time variations and improved tracking.
- *Cost Reduction:* Identified high-cost routes and suggested cost-efficient alternatives.
- *Inventory Health:* Monitored stock levels to prevent stock-outs and overstocking.

## Tools Used
- *Power BI:* For Interactive Dashboards.
- *Power Query:* For Data Cleaning and ETL.
- *DAX:* For advanced measures like OTD% and Lead Time.
- *Dataset:* Global Logistics CSV/Excel.
